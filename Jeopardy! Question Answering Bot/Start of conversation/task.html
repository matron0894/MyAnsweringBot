Description

User greeting is the face of our question answering system. When users start a conversation with a QA bot, they want to know what the system can do. To personalize such bots, developers give them names (for example, Alexa, Siri, Cortana, and so on).

In this stage, teach your bot to greet a user and ask for their request. Your QA bot should say their name (you can choose any name you like). After greeting, the bot should ask for a user request and take it.

Objectives

In this stage, your program should:

  1. Print a greeting message: Hello! I'm (the QA bot name you chose), a question answering bot who knows answers to all questions from the 'Jeopardy!' game.
  2. Ask a user to input their request with the Ask me something! string and read the query;
  3. Print Let's play! after reading user input to make it clear the bot has taken the request.
Example

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1: the QA bot introduces themselves and the user inputs their query

Hello! I'm Jane, a question answering bot who knows answers to all the questions from the 'Jeopardy!' game.
Ask me something!
> Get me a "Noah's boy on bread", one of these lunch favorites.
Let's play!
